# my_customs
This project can be used for Mac osx Iterm2 & terminal with bash or zsh only
SHELL custom configs:
BASH
**********************************************
Usage: Add below into $HOME/.zshrc
```bash
source $MYRUNTIME/customs/my_shell/rcfile
**********************************************
ZSH
```
Usage: Add below into $HOME/.bashrc
```zsh
source $MYRUNTIME/customs/my_shell/rcfile
```
**********************************************

	my.itermcolors
